Markdown Extra
==============

**markdown-extra** is a set of `python-markdown <https://pypi.python.org/pypi/Markdown>`_ extensions.

Installation is done with pip.

.. code:: shell-session

    $ pip install markdown-extra

This package is intended to run in both python 2.7 and 3.3+.
